# Ask Lizzie

Ask Lizzie when you're not sure.

A mobile-first web app that helps people understand everyday paperwork and respond with confidence. Take a photo of a letter, get a clear explanation, copy a reply.

---

## Run locally

```bash
npm install
# Create .env.local with: ANTHROPIC_API_KEY=sk-ant-xxxxx
npm run dev
```

## Deploy to Netlify

Push to GitHub, then import at [app.netlify.com](https://app.netlify.com). Add `ANTHROPIC_API_KEY` in the environment settings under Site Settings > Environment Variables.
